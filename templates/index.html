What is Federated Learning?

Federated Learning is a form of Machine Learning where training is localised on the client's device. The model is trained across these multiple devices, but the server only aggregates the model updates. This helps in preserving user privacy and data security.

This website will hopefully help showcase the pros and cons with both Federated Learning and it's surrounding Privacy Enhancing Technologies.
